# Buggy Bar Build Book

This document describes how to build a Buggy Bar, a kite control bar designed by and for kite buggiers. Philip Chase is the principal designer of the Buggy Bar and principal author of this document. 

To see the rendered document, see http://philipbchase.com/

## Contributions

Contributions, corrections and questions are welcome. Please open an issue, submit a pull request or just email me directly.

This book is written using [Quarto](https://quarto.org/docs/books/). It might be a lot to bite off just to point out typos, so please feel free to just open an issue and cut and paste the bad text into the issue.
